## Welcome!

The Wazuh Indexer is a highly scalable, full-text search and analytics engine. This Wazuh central component indexes and stores alerts generated by the Wazuh server and provides near real-time data search and analytics capabilities.

Wazuh Indexer is an open source fork of [OpenSearch](https://github.com/opensearch-project/opensearch).

This repository stores the source code of the Wazuh Indexer Notifications Plugin and its configurations.

## Wazuh Indexer Notifications Plugin

Wazuh Indexer Notifications allows users (engineers, including but not limited to developers, DevOps, IT Engineer, and IT admin) to manage and send notifications from the Wazuh Indexer through various communication channels. It helps automate the process of delivering alerts and system events on an on-demand or a periodical basis. Further, it also automates the process of notifying various stakeholders within the organization when specific conditions are met. These stakeholders include but are not limited to, executives, managers, and engineers (developers, DevOps, IT Engineer) who can receive updates via channels such as email, Slack, Amazon Chime, and custom Webhooks. In order to configure, manage, and receive these notifications, users should have the necessary permissions as defined under Roles and Privileges.
